🎮 猜数字积分小游戏
 
✨ 游戏简介
 
一款简约趣味控制台猜数字休闲小游戏

内置本地积分存档系统，猜题次数越少奖励积分越高

支持自定义数字区间，数据永久本地保存，轻松休闲娱乐
 
 
 
📌 核心游戏功能
 
🧮 积分获取规则
 
- 1次精准猜对 → 奖励 10积分
- 2次猜对 → 奖励 9积分
- 依次逐次递减
- 十次及以上猜对 → 固定奖励 1积分
 
💾 本地存档系统
 
- 程序启动自动读取历史存档积分
- 每局游戏结束自动保存积分数据
- 关闭程序、重启电脑积分均不会清空丢失
 
🎯 游玩模式选择
 
1. 快速开局：默认 1~100 标准猜数模式
2. 高级自定义：自由设置猜数字最大值与最小值
 
📋 对局结束选项
 
🔹 1. 再来一局，继续游玩
🔹 2. 结束游戏，直接退出
🔹 3. 实时查看当前所持总积分
 
 
 
🎮 游戏游玩教程
 
1. 双击运行  猜数字游戏.exe  启动程序

2. 进入主界面，选择对应游玩模式

3. 根据控制台提示输入猜测数字

4. 系统自动提示数字偏大/偏小，自动缩小猜测范围

5. 成功猜对后自动结算本局积分并自动存档

6. 自由选择继续游戏或退出程序即可
 
 
 
📁 程序文件明细
 
plaintext
  
猜数字游戏.exe      ｜  主程序发行启动文件

积分.txt            ｜  自动生成积分数据存档文件
 
 
⚠ 重要提示
 
1. 请勿手动篡改  积分.txt  内部数据
2. 直接删除  积分.txt  文件，即可一键清空全部游戏积分
